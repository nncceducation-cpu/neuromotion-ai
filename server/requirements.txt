
# =============================================================================
# NEUROMOTION AI - BACKEND DEPENDENCIES & INSTALLATION GUIDE
# =============================================================================
#
# CRITICAL: GPU-accelerated libraries (PyTorch, MMCV) depend heavily on your 
# specific hardware drivers (CUDA version) and OS. 
#
# NOTE FOR CUDA 13.1 USERS:
# NVIDIA drivers are backward compatible. You should install the PyTorch wheels 
# built for CUDA 12.4 (the latest stable release), which will run perfectly 
# on your 13.1 driver.
#
# Follow the steps below in order using PowerShell.
#
# -----------------------------------------------------------------------------
# STEP 1: BASIC DEPENDENCIES (PowerShell)
# -----------------------------------------------------------------------------
# Run this command to install the web server and data processing tools:
#
# pip install fastapi uvicorn python-multipart numpy scipy opencv-python-headless google-genai
#
# -----------------------------------------------------------------------------
# STEP 2: PYTORCH (For CUDA 13.1 Driver)
# -----------------------------------------------------------------------------
# We use the CUDA 12.4 wheels.
#
# OFFICIAL DOWNLOAD SELECTOR: 
# https://pytorch.org/get-started/locally/
#
# COMMAND (PowerShell):
# pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
#
# Verify after install: 
# python -c "import torch; print(torch.cuda.is_available())"
#
# -----------------------------------------------------------------------------
# STEP 3: OPENMMLAB / VITPOSE (Computer Vision Stack)
# -----------------------------------------------------------------------------
# This stack requires 'mmcv'. The 'openmim' manager usually handles version matching.
#
# A. Install MIM (OpenMMLab Package Manager):
# pip install openmim
#
# B. Install MM Engine:
# mim install mmengine
#
# C. Install MMCV (The Critical Step):
# WARNING: If pre-built wheels aren't found for the newest PyTorch/CUDA combo, 
# MIM might try to compile from source (requires Visual Studio C++ Build Tools).
#
# OFFICIAL GUIDE: https://mmcv.readthedocs.io/en/2.x/get_started/installation.html
#
# Recommended Command:
# mim install "mmcv>=2.0.0"
#
# D. Install MMPose (The Model Library):
# OFFICIAL GUIDE: https://mmpose.readthedocs.io/en/latest/installation.html
# mim install "mmpose>=1.0.0"
#
